# Dependências

## Backend (Go)
- Go versão 1.x
- [Gin](https://github.com/gin-gonic/gin) - Framework web para Go
- [Gorm](https://gorm.io) - ORM para Go
- [PostgreSQL](https://www.postgresql.org/) - Banco de dados utilizado

## Frontend (Next.js)
- Node.js
- Next.js
- React
- Tailwind CSS
